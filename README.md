## 🚀 About the Project

This is the personal portfolio of **Obadiah Louis Adamu**, a passionate Data Scientist & AI Engineer. The project is designed to showcase skills, professional experience, and technical projects in a sleek, fast, and fully responsive web interface. 

It uses modern frontend tools and is deployed seamlessly via Vercel.

## ✨ Features

- **Component-Driven:** Built modularly with React to easily update sections independently.
- **Lightning Fast:** Powered by Vite for rapid development and optimized production builds.
- **Responsive Design:** Styled heavily with Tailwind CSS to look perfect on mobile, tablet, and desktop.
- **Dynamic Scroll:** Utilizes Intersection Observers for smooth transitions as you scroll through the page.

## 💻 Local Development

Want to run this project locally to test changes? Follow these simple steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/lobadiah/obadiah-portfolio.git
   cd obadiah-portfolio
   ```

2. **Install all dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **View the site:**
   Open your browser and navigate to `http://localhost:5173`. 

## 🌐 Deployment

This application is deployed on **Vercel**. To update the live site, ensure you have the Vercel CLI installed and run:

```bash
npx vercel --prod
```

It will automatically package your code and push it to production!
